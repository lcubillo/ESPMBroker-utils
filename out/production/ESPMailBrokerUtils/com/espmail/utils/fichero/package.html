Contiene Fichero.java que es una clase abstracta y la clase Propiedades.java.

Fichero.java es una clase abstracta que permite comprobar cada cierto tiempo un archivo y ver si ha cambiado. Si se hubiera modificado hace lo que se defina en el método reload.

Propiedades.java permite recargar el fichero que permite recargar cualquier archivo .properties que se le pase. 

ej.-
	Propiedades prop = new Propiedades(a.properties);
